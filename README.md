# exchanges

This repository is a **learning lab project** designed to help develop practical skills relevant to working with complex telephone exchange systems.

## Getting Started

This project provides step-by-step documentation to set up and operate a SUSE Linux server for learning system administration, databases, and telephony concepts.

### Documentation Structure

Follow the guides in order:

| Step | Document | Description | Est. Time |
|------|----------|-------------|-----------|
| **Setup** | | | |
| 01 | [Prerequisites](doc/01-prerequisites.md) | Check system requirements, enable Hyper-V | 15-30 min |
| 02 | [VM Quick Start](doc/02-vm-quickstart.md) | **Recommended:** Import pre-built VM (~192 MB download) | 10-15 min |
| 02(b) | [VM Setup (Manual)](doc/02-vm-setup.md) | **Alternative:** Manual OS installation (~4.2 GB download) | 30-60 min |
| **Access** | | | |
| 03 | [SSH Access](doc/03-ssh-access.md) | Configure SSH from Windows, optional VS Code setup | 15-20 min |
| 04 | [SSH Hardening](doc/04-ssh-hardening.md) | **Important:** Secure your SSH configuration | 10-15 min |
| 05 | [System Configuration](doc/05-system-config.md) | Configure system, learn essential commands, resources | 20-30 min |
| **Future** | | | |
| 06 | PostgreSQL (or other DB) Setup | Install and configure database *(coming soon)* | TBD |
| 07 | SQL Learning | Create a simple and fun database *(coming soon)* | TBD |
| 08 | Plex Media Server | Home media streaming setup *(coming soon)* | TBD |
| 09 | VoIP Lab | Telephony concepts with Asterisk (or similar) *(coming soon)* | TBD |

**Total setup time:** ~1-2 hours depending on path chosen

### Quick Start Path

For the fastest setup:
1. Complete [01 Prerequisites](doc/01-prerequisites.md)
2. Follow [02 VM Quick Start](doc/02-vm-quickstart.md) (pre-built VM)
3. Set up [03 SSH Access](doc/03-ssh-access.md)
4. Apply [04 SSH Hardening](doc/04-ssh-hardening.md)
5. Configure [05 System](doc/05-system-config.md) and explore learning resources
